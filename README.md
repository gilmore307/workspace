# workspace

Primary OpenClaw workspace for host-level operating rules, memory, and assistant configuration.

## Contents

- `AGENTS.md` — workspace operating rules
- `SOUL.md` — assistant persona and stance
- `USER.md` — stable user preferences
- `TOOLS.md` — local tool and secret alias notes
- `memory/` — durable memory and oversight records

This repository is not a product codebase. It is the working control surface for OpenClaw on this host.
